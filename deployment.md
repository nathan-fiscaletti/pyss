# Deployment

1. Build the project

   ```bash
   $ python3 -m build
   ```
   ```
   dist/
    ├── pyss-X.X.X-py3-none-any.whl
    └── pyss-X.X.X.tar.gz
   ```

2. Upload to testpypi
   ```bash
   $ python3 -m twine upload --repository testpypi dist/*
   ```
